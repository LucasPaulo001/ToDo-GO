# ToDo-GO
Backend de um To-Do List desenvolvido em Golang, criado para fins de estudo e prática com APIs REST.

## Funcionalidades
- Criar tarefa
- Listar tudo/uma
- Editar tarefa
- Excluir tarefa

## Tecnologias
[Golang](https://go.dev/) => linguagem principal

[Gin Gonic](https://gin-gonic.com/) => framework para criação de APIs REST

[GoDotEnv](https://github.com/joho/godotenv) => para gerenciar variáveis de ambiente

> OBS: Dados armazenados na memória
